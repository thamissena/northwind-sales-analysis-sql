# Northwind Sales Analysis with SQL

This project performs exploratory analysis and business queries using the **Northwind SQLite database**.
The objective is to practice SQL concepts such as filtering, aggregation, joins, and data exploration to extract insights from sales data.

This case study was developed as part of the **Lighthouse preparatory course by Indicium Academy**.

---

## Dataset

The project uses the classic **Northwind sample database**, which contains information about products, categories, orders, customers, and sales transactions.

The SQLite database file is available in this repository.

## SQL Analysis

The following analyses were performed using SQL:

1. List of all products and their prices.
2. Identification of products costing more than 20 with fewer than 10 units in stock.
3. Calculation of total sales and average product price.
4. Identification of category names associated with CategoryID.
5. Identification of the top 3 product categories that generated the highest total revenue.
